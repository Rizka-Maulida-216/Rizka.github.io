HTML

Bahasa markup yang digunakan untuk membuat halaman web. HTML adalah markup yang digunakan untuk mendefinisikan struktur halaman web. Seperti membuat elemen paragraf, gambar, link,dll.

CSS

CSS adalah bahasa yang digunakan untuk mengatur tampilan elemen-elemen HTML. Dengan CSS, kita dapat memberikan warna background, border, ukuran font, mengatur posisi elemen, dll.

JS

Javascript yaitu bahasa pemrograman(berjalan pada web browser) yang digunakan untuk membuat halaman web yang lebih interaktif. Beberapa fungsi yang bisa digunakan yaitu menampilkan elemen tertentu hanya pada browser saja, membuat fitur yang memungkinkan pengguna bisa menyesuaiakan ukuran font.

PHP

PHP adalah bahasa pemrograman server side scripting yang bersifat open source. Bahasa pemrograman ini banyak digunakan untuk pengembangan website.

Python

Python adalah bahasa pemrograman interpretatif yang dapat digunakan di berbagai paltform dengan filosofi perancangan yang berfokus pada tingkat keterbacaan kode dan merupakan salah satu bahasa populer yang berkaitan dengan Data Science,dan Machine Elearning.
